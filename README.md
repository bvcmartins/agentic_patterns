# Agentic Patterns 

This is my study repo for everything Agentic AI. I use it to test different architectures and how the agents will perform with different tools. I am particularly interested in interactions with SQL and Neo4j, which require the model to master advanced querying techniques. 

## Description

For this project we have been collecting ideas and code from multiple sources like Langchain, Medium and Hugging Face. I try to refer to the original source of the code on the first cell of the notebooks. Of course I tend to change the code quite a lot and to test different applications and data sources. 

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bvcmartins/agentic_patterns.git
   ```

2. Install dependencies:
   ```bash
   conda install -y --file requirements.txt
   ```
